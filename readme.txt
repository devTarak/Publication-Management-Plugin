== Description ==
A custom plugin to manage the "Publication" custom post type with PDF uploads. Features include a dedicated post type, PDF meta box with uploader, and frontend PDF viewer.

== Installation ==
1. Upload the `publication-management` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
